## 8. Grab the review short-link (do this the same sitting)

1. Open the profile in Google Search (search "Delta Roe Elk Grove" while signed
   in as the owner) or at business.google.com.
2. Click **"Ask for reviews"** (sometimes shown as **Share review form** under
   the profile menu).
3. Copy the short link — it looks like `https://g.page/r/XXXXXXXXXX/review`.
4. Paste it into these four places:
   - `lib/site.ts` → `REVIEW_URL` constant (replaces the `REPLACE_ME`
     placeholder; this powers the gold button on deltaroe.com/review)
   - `docs/review-templates.md` → every `[REVIEW-LINK]` placeholder
   - Facebook page "Review us on Google" pinned post (optional)
   - Nowhere else needed — the printed QR card points at deltaroe.com/review,
     which forwards visitors to this same link, so **printed cards never go
     stale** even if Google changes the short-link format.
